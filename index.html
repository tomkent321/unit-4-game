Instructions:

  • At the start of each game you will be given a target number, a current score and four crystals.
  • Each crystal has a randomly assigned value that is not visible.
  • As you click on a crystal the hidden value will be added to the current score. Remember each crystal has a different hidden value. You can determine the value of that crystal by subtracting the new current value from the previous current value.
  • You win by hitting the target number exactly.
  • You lose if you go over the target number.
